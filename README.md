A high-end, interactive professional portfolio built with **React**, **Tailwind CSS**, and **Framer Motion**. This site showcases my expertise in building secure, scalable enterprise applications using **Django** and **React**.

## ✨ Key Features
- **Cinematic About Section**: Interactive scroll animations where images and text slide apart smoothly.
- **Dynamic Project Showcase**: Hover-active project cards featuring TechNexus, VSCMS, and AI logic.
- **Functional Contact System**: Direct-to-email messaging integrated via Formspree.
- **Live Progress Tracking**: A sleek, gradient scroll bar that follows user reading progress.

## 🚀 Local Setup
1. Clone the repo: `git clone https://github.com/jayasoorya-kr/jayasoorya-portfolio.git`
2. Install dependencies: `npm install --legacy-peer-deps`
3. Run dev server: `npm run dev`
